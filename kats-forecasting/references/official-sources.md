# Official Sources Consulted

Use these sources to verify Kats behavior. Do not invent APIs, class names, parameters, or capabilities that are not present in official docs/source.

- GitHub repository and README: https://github.com/facebookresearch/Kats
- Kats homepage: https://facebookresearch.github.io/Kats/
- PyPI package page: https://pypi.org/project/kats/
- Tutorials directory: https://github.com/facebookresearch/Kats/tree/main/tutorials
- Forecasting tutorial notebook: https://github.com/facebookresearch/Kats/blob/main/tutorials/kats_201_forecasting.ipynb
- Global model tutorial notebook: https://github.com/facebookresearch/Kats/blob/main/tutorials/kats_205_globalmodel.ipynb
- API index: https://facebookresearch.github.io/Kats/api/
- `TimeSeriesData` API: https://facebookresearch.github.io/Kats/api/kats.consts.html
- Forecasting models API index: https://facebookresearch.github.io/Kats/api/kats.models.html
- Base model API: https://facebookresearch.github.io/Kats/api/kats.models.model.html
- Model APIs reviewed: ARIMA, Bayesian VAR, harmonic regression, Holt-Winters, linear model, LSTM, Prophet, quadratic model, SARIMA, STLF, Theta, VAR.
- Ensemble APIs reviewed: base ensemble, Kats ensemble, median ensemble, weighted average ensemble.
- Temporal hierarchical reconciliation API: https://facebookresearch.github.io/Kats/api/kats.models.reconciliation.thm.html
- Meta-learning APIs reviewed: `kats.models.metalearner` package pages.
- Source tree for model modules not present in Sphinx API: https://github.com/facebookresearch/Kats/tree/main/kats/models
- Source files reviewed for source-visible models: `ml_ar.py`, `neuralprophet.py`, `simple_heuristic_model.py`, and `globalmodel/`.
