Consultation notes:

- GitHub README and docs identify ADTK as an unsupervised/rule-based time-series anomaly detection toolkit with detectors, transformers, aggregators, pipes, data utilities, metrics, and visualization.
- The latest stable release found in official package/release pages is `0.6.2`, released April 17, 2020.
- The official examples list detector support for threshold, quantile, IQR, generalized ESD, persistence, level shift, volatility shift, seasonal, autoregression, clustering/outlier wrappers, regression, PCA, and custom detectors.
- The documentation does not present ADTK as a native optimal changepoint segmentation library or a supervised changepoint library.
