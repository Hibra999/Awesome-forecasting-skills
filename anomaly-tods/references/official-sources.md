Consultation notes:

- README describes TODS as a full-stack automated machine-learning system for outlier detection on multivariate time-series data.
- README/docs document three scenarios: point-wise, pattern-wise, and system-wise outlier detection.
- Hosted docs list the D3M-style primitive modules and detection algorithms; the entry point file adds several primitives not surfaced in the hosted index.
- PyPI lists `tods 0.0.2`, released September 21, 2020.
