Notes:
- Stable docs identify ETNA 3.0.0.
- The official installation page spells the classification extra as `classiciation`; this skill preserves that as a documented caveat and does not use classification for forecasting.
- LightGBM and XGBoost wrapper classes were not listed in the browsed ETNA 3.0 model API reference; do not claim those class names without checking a specific installed version/source.
